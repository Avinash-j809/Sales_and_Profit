# Stock Analysis and Dashboard Project

## Overview
This repository contains detailed analysis and visualizations of Amazon stock data from 1997 to recent years, along with an interactive dashboard. The project includes raw data, summarized insights, and visual representations to facilitate understanding stock performance trends over time.

## Files Included
- **Amazon-Stock-Analysis.xlsx**: Contains raw stock data, pivot summaries, charts, and dashboards analyzing Amazon's stock prices and trading volumes.
- **S-0P-dashboard.xlsx**: An interactive dashboard file that provides visual summaries, key metrics, and insights into Amazon stock behaviors, trends, and other financial indicators.

## How to Use
- Open the Excel files in Microsoft Excel or compatible tools like Google Sheets.
- Review the raw data for detailed day-to-day stock price movements.
- Use the pivot tables and charts for a summarized view of long-term trends.
- Explore the dashboard for quick insights and visual trends of Amazonâ€™s stock performance.
- Export data or charts for further analysis using Python, R, or BI tools.

## Key Features
- Extensive historical data from 1997 onwards.
- Summary tables with average prices, volume, and trend analysis.
- Visualizations showing price fluctuations, volume, and other key metrics.
- Interactive dashboards for rapid understanding of stock trends.
